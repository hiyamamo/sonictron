# Sonictron
Music player for [Subsonic](http://www.subsonic.org/) with electron  

![screenshot](images/screenshot.png)  

## Build & Run

```
 # clone repository
 git clone https://github.com/hiyamamo/sonictron.git
 
 # install dependencies
 npm install
 
 # build
 npm run build

 # run
 npm start
```
